# Logical Front - Next.js Application

A modern React application built with Next.js 15, featuring enterprise IT infrastructure solutions.

## Technology Stack

- **Next.js 15** with App Router
- **React 18** with TypeScript
- **Tailwind CSS** for styling
- **Vitest** for testing
- **ESLint** for code quality

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
- `npm test` - Run tests
- `npm run test:ui` - Run tests with UI
- `npm run test:coverage` - Run tests with coverage

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `src/components/` - React components organized by atomic design
- `src/types/` - TypeScript type definitions
- `src/utils/` - Utility functions
- `branding/` - Static branding assets
- `docs/` - Project documentation
- `public/` - Static assets

## Features

- Server-Side Rendering (SSR)
- Static Site Generation (SSG)
- Responsive design with Tailwind CSS
- Atomic design component architecture
- Comprehensive SEO metadata
- Interactive mega navigation
- Modern TypeScript development
