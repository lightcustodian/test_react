# Logical Front Enterprise Visual Redesign - CE System Prompt

## PROJECT OVERVIEW
Transform Logical Front's website visual design to project enterprise-class capabilities while maintaining family values and relationship-focused culture. This is a comprehensive visual redesign project using the Context Engineering system.

---

## BUSINESS TRANSFORMATION CONTEXT

### **Strategic Shift**
- **FROM**: Hardware/software reseller with installation/support services  
- **TO**: Managed private cloud provider (maintaining traditional services)
- **Goal**: Position alongside larger, established companies in the industry
- **Target Perception**: Enterprise capabilities with boutique attention

### **Target Market**
- **Segments**: Higher education, K-12 school districts, regional healthcare, manufacturing, business
- **Size**: Mid-market ($250K-$2M IT budgets, 100-1,000 employees)
- **Geography**: Arkansas (AR-EON advantage), Utah, New Mexico
- **Value Proposition**: 33% cost savings vs public cloud with predictable pricing

### **Brand Preservation**
- **Colors**: Blue (#125EAD) and Green (#4BB74E) - must be incorporated
- **Culture**: Family business personality with professional competence
- **Differentiation**: Enterprise-grade performance with relationship focus

1. **Professional Typography**: Clean, readable hierarchy that projects expertise

3. **Enterprise Color Usage**: Sophisticated application of blue/green palette
4. **Trust-Building Elements**: Visual cues that suggest scale and reliability
5. **Technical Depth Display**: Visual methods to showcase infrastructure capabilities

### **Competitive Differentiation**
- **Warmer Personality**: Green accents vs typical all-blue tech companies
- **Industry-First Approach**: Show industry expertise prominently
- **Relationship Focus**: Visual elements that suggest partnership over transactions
- **Arkansas Advantage**: Subtle geographic positioning elements

- **Sophisticated Warmth**: Professional expertise with approachable design
- **Family Business**: Friendly micro-interactions and human elements
- **Technical Competence**: Clean, modern interface suggesting infrastructure expertise
- **Enterprise Scale**: Visual cues that suggest larger company capabilities
